The preferred icon library is Google's <a href="https://design.google.com/icons/">Material icons</a>. We include the entire library in the UI Components project in the form of svg paths that get inlined into your markup.

Use these icon classes on `<span>` elements wrapping your inline svg icons. Here is a simple icon in it's purest form.

```example
<span class="c-icon c-icon--large">
  <svg xmlns="http://www.w3.org/2000/svg" width="24" height="24" viewBox="0 0 24 24"><path d="M15 1H9v2h6V1zm-4 13h2V8h-2v6zm8.03-6.61l1.42-1.42c-.43-.51-.9-.99-1.41-1.41l-1.42 1.42C16.07 4.74 14.12 4 12 4c-4.97 0-9 4.03-9 9s4.02 9 9 9 9-4.03 9-9c0-2.12-.74-4.07-1.97-5.61zM12 20c-3.87 0-7-3.13-7-7s3.13-7 7-7 7 3.13 7 7-3.13 7-7 7z"/></svg>
</span>
```
```html
<span class="c-icon"><svg>...</svg></span>
```
